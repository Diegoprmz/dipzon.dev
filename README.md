# DIPZON.DEV  🚀 

**Sitio Web profesional para servivios de desarrollo web, consultoría estrátegica empresarial, automatizaciones y más.**

La wewb será diseñada en un pincipio con: `HTML`, `SASS`, `JavaScript` y será potenciado con `Next.js`.
Este proyecto está pensado para escalar con el tiempo y evolucionar hacía una app web más completa con funcionalidades que ya iré contando.

---

## Tecnologías usadas 🛠️

- [Next.js](https://nextjs.org/) – Framework de React para frontend y backend
- [SASS](https://sass-lang.com/) – Preprocesador CSS
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) – Lenguaje de programación principal
- HTML5 Semántico
